# 🗳️ Voting System

A simple and responsive Voting System web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- User-friendly interface
- Vote for candidates
- Real-time vote counting
- Responsive design
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

voting-system/
│── index.html
│── style.css
│── script.js
│── README.md

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start voting!

## 📸 Future Improvements

- Add login authentication
- Store votes in database
- Admin dashboard
- Results chart visualization
